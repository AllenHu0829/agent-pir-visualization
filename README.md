# PIR 可视化打点工具

PIR（被动红外传感器）触发测试的可视化坐标工具。上传测试数据表格后自动生成以设备为原点的二维打点图。

## 使用

1. 打开在线地址或本地 `index.html`
2. 上传 CSV / Excel 文件（包含距离、角度、触发状态）
3. 自动生成可视化坐标图

## 数据格式

| 距离(m) | 角度(°) | 触发 |
|---------|---------|------|
| 3       | 0       | 是   |
| 5       | 30      | 否   |

字段名支持中英文自动匹配。

## 技术栈

- HTML + CSS + Vanilla JS
- Chart.js（散点图）
- PapaParse（CSV 解析）
- SheetJS（Excel 解析）
